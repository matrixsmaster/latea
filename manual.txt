**********************************************************************
* 1. INTRODUCTION                                                    *
**********************************************************************

Latea is a small graphical text editor with built-in autocomplete
support. Autocomplete can be disabled, driven by a word list, learned
from the current document, supplied by a llama.cpp-compatible server,
run through the embedded AI engine, or generated by a Markov model.


**********************************************************************
* 2. FILE MENU                                                       *
**********************************************************************

New                       Ctrl+N
  Starts an empty document. If the current document has unsaved
  changes, Latea asks whether to save, discard, or cancel.

Open...                   Ctrl+O
  Opens a file using the native file chooser. Unsaved changes are
  handled before the file is loaded.

Save                      Ctrl+S
  Saves the current document. If the document has no filename yet,
  this command opens the Save As dialog.

Save As...                Ctrl+Shift+S
  Saves the document under a new filename. If the target file already
  exists, Latea asks before overwriting it.

Quit                      Ctrl+Q
  Closes Latea. Unsaved changes are handled before the program exits.


**********************************************************************
* 3. EDIT MENU                                                       *
**********************************************************************

Undo                      Ctrl+Z
  Reverses the last edit recorded by Latea's history system.

Redo                      Ctrl+Y
  Reapplies an edit that was undone.

Find                      Ctrl+F
  Opens the Find dialog. Enter text and press Find Next to search from
  the current caret position. Search wraps around to the top.

Replace                   Ctrl+R
  Opens the Find / Replace dialog. Replace changes the selected match
  or the next match. Replace All replaces every match, starting at the
  caret and wrapping around.

Preferences...            Ctrl+,
  Opens the Preferences dialog. See chapter 8.


**********************************************************************
* 4. VIEW MENU                                                       *
**********************************************************************

Font...
  Opens the Choose Font dialog. This dialog controls the editor font,
  text size, text color, ghost suggestion color, background color, and
  selection color.

Word Wrap                 F6
  Toggles wrapping at the editor bounds.

Line Numbers
  Toggles the line number margin.


**********************************************************************
* 5. TOOLS MENU                                                      *
**********************************************************************

Stats                     Ctrl+I
  Shows document statistics: total lines, maximum line length, word
  count, character count, byte count, and a Tokens button for counting
  model tokens.

Auto-indent
  When enabled, pressing Enter copies leading spaces and tabs from the
  current line. If Tabs as spaces is set, Backspace removes
  indentation in tab-width chunks when the caret is still inside
  leading spaces.


**********************************************************************
* 6. AUTOCOMPLETE MENU                                               *
**********************************************************************

Tools / Autocomplete / Trigger        Ctrl+Space
  Requests an autocomplete suggestion immediately.

Tools / Autocomplete / Clear          Esc
  Clears the visible suggestion. Esc also cancels a running AI
  request.

Tools / Autocomplete / Reset          Ctrl+F1
  Cancels all autocomplete work, stops the external llama server,
  clears embedded AI state, and clears the visible suggestion.

Tools / Autocomplete / Continuous     F8
  When enabled, Latea schedules suggestions automatically after text
  changes. The delay is controlled by the Delay ms preference.

Tools / Autocomplete / Type
  Selects the autocomplete engine:

  Disabled                            F1
    No suggestions are produced.

  Dictionary File                     F2
    Reads words from the Dictionary path file. When the current word
    is a prefix of a longer dictionary word, Latea suggests the
    remaining characters.

  Current File                        F3
    Scans words in the current document and suggests completions that
    match the current word prefix.

  AI Connector                        F4
    Sends context to an external llama.cpp-compatible server. Latea
    can launch the server if a llama server path is configured.

  Embedded AI                         Shift+F4
    Uses Latea's embedded model engine and the configured model path.
    The status line reports token and context usage while generating.

  Markov model                        F5
    Builds a lightweight model from the current document and suggests
    probable continuations. This mode is local and updates as the text
    changes.


**********************************************************************
* 7. SUGGESTION KEYS                                                 *
**********************************************************************

Right Arrow
  Accepts the full visible suggestion.

Ctrl+Right Arrow
  Accepts only the next word-sized part of the visible suggestion. The
  remaining text stays visible as a new suggestion.

Down Arrow
  Moves to the next suggestion variant.

Up Arrow
  Moves to the previous suggestion variant.

Esc
  Clears the suggestion. If an AI request is busy, Esc cancels pending
  autocomplete work.


**********************************************************************
* 8. PREFERENCES                                                     *
**********************************************************************

Preset
  Selects a saved preference preset. "Latea" is the main preset and
  cannot be deleted. New creates a preset from the current settings.
  Delete removes the selected preset, except for "Latea".

Dictionary path
  Word-list file used by Dictionary File autocomplete. The file is
  read as whitespace-separated words.

Max suggestion
  Maximum suggestion length for dictionary and current-file modes. It
  also limits generation length for AI and Markov modes.

Delay ms
  Delay before continuous autocomplete triggers after editing.

Model path
  Model file used by Embedded AI and by the launched AI connector
  server.

llama server path
  Path to the llama.cpp server executable used by AI Connector. If not
  configured, user-managed server will be implied.

AI host
  Hostname or IP address for the external AI connector. Default is
  127.0.0.1.

port
  TCP port for the external AI connector. Default is 8080.

Endpoint mode
  Completion Only always uses the completion endpoint.
  Prefer Infill uses the infill endpoint when suffix context is
  available; otherwise it falls back to completion.

Timeout
  AI connector request timeout in milliseconds.

AI slot id
  Slot id sent to the external server. Used to separate multiple
  active users simultaneously accessing the AI server (if needed).
  Normally is set to 0.

Use prompt cache
  Enables prompt-cache use for AI requests when supported by the
  backend.

Context length
  Context window size for AI generation (both embedded and external).

TurboQuant
  Enables TurboQuant KV cache in Embedded AI engine. Allows using
  bigger contexts on systems with limited memory.

Prefix chars
  Maximum number of characters before the caret sent as prefix
  context.
  Setting to 0 uses the whole text.

Suffix chars
  Maximum number of characters after the caret sent as suffix context.
  Suffix context is needed for infill requests.

Temperature
  AI sampling temperature. Lower values are more deterministic.

Top P
  Nucleus sampling cutoff. The AI considers the smallest set of likely
  next tokens whose combined probability reaches this value. Lower
  values narrow the choice set and make output steadier; higher values
  allow more variety.

Top K
  Limits sampling to the K most likely next tokens. Lower values make
  suggestions more conservative; higher values allow less common
  tokens. Setting this too high can make suggestions noisier.

System prompt
  System prompt used for AI suggestions.

Tabs as spaces
  If greater than zero, pressing Tab inserts this many spaces. With
  Auto-indent enabled, Backspace removes leading indentation by this
  width.

Save backup
  Before saving over an existing file, renames the current on-disk
  file to the same filename with ".bak" appended, then saves normally.

Stop on punctuation
  In Markov mode, stops a suggestion when punctuation is reached.


**********************************************************************
* 9. FONT AND COLOR DIALOG                                           *
**********************************************************************

The Choose Font dialog lists installed fonts and available sizes. The
size can also be typed directly.

Text
  Color used for normal document text.

Ghost
  Color used for inline autocomplete suggestions.

Background
  Editor background color.

Selection
  Selection highlight color.

**********************************************************************

Models
  Shows architectures and model capabilities supported by this build.

About
  Shows the program name and version text.
